# Chatbot with LLM & RAG

Это небольшой проект чат-бота, который демонстрирует сочетание **Large Language Model (LLM)** и техники **Retrieval-Augmented Generation (RAG)**. Бот умеет извлекать информацию с указанных веб-сайтов и использовать её как контекст для ответов на вопросы.

## О проекте

Основная идея проекта — создание интеллектуального ассистента, который может отвечать на вопросы, основываясь не только на своих внутренних знаниях, но и на актуальной информации с заданных вами веб-страниц. Это реализовано с помощью библиотеки LangChain, которая объединяет все этапы: от загрузки и обработки текста до взаимодействия с LLM.

Проект был создан для изучения и демонстрации подхода RAG, который позволяет эффективно подключать к языковым моделям внешние источники знаний.

## Как это работает?

Процесс работы бота можно разделить на два основных этапа:

1.  **Подготовка базы знаний (Индексация)**:
    *   **Scraping**: С помощью инструментов из директории `scrap` извлекается текст с целевых веб-сайтов.
    *   **Chunking & Embedding**: Текст разбивается на фрагменты (chunks), и для каждого фрагмента создаются векторные эмбеддинги (числовые представления смысла текста).
    *   **Сохранение**: Векторные эмбеддинги сохраняются в векторную базу данных FAISS (директория `faiss_db`) для быстрого поиска в будущем.

2.  **Ответ на вопрос (Генерация)**:
    *   Когда пользователь задаёт вопрос, система преобразует его в такой же векторный эмбеддинг.
    *   Выполняется поиск в FAISS по семантической близости, чтобы найти фрагменты из базы знаний, наиболее релевантные вопросу.
    *   Найденные фрагменты добавляются в промпт для LLM в качестве контекста.
    *   LLM (Large Language Model) формирует финальный ответ, опираясь на предоставленный контекст.

## Установка и запуск

Следуйте этим шагам, чтобы запустить проект локально.

### Предварительные требования
*   Python 3.8 или выше
*   `pip` (менеджер пакетов Python)

### Шаги по установке

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/kirimusha/Chatbot-with-LLM-RAG.git
    cd Chatbot-with-LLM-RAG
    ```

2.  **Создайте и активируйте виртуальное окружение** (рекомендуется):
    ```bash
    python -m venv venv
    source venv/bin/activate  # Для Linux/macOS
    # venv\Scripts\activate   # Для Windows
    ```

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Настройте переменные окружения:**
    Вам потребуется указать ваш API-ключ для доступа к LLM (например, OpenAI). Создайте файл `.env` в корне проекта и добавьте туда:
    ```env
    OPENAI_API_KEY=ваш_секретный_ключ
    ```
    *(Примечание: точное название переменной может отличаться в зависимости от используемой модели, проверьте код в `app.py` или модулях `rag/`.)*

5.  **Запустите приложение:**
    ```bash
    python app.py
    ```
    После этого бот должен запуститься и быть готов к работе. Способ взаимодействия (веб-интерфейс или консоль) зависит от реализации в `app.py`.

## Использование

1.  **Добавление знаний:** Для того чтобы бот мог отвечать на вопросы по конкретному сайту, необходимо сначала запустить процесс извлечения данных из него. Это, вероятно, делается с помощью скриптов в папке `scrap`.
2.  **Общение:** После загрузки данных в векторную базу (`faiss_db`), вы можете задавать вопросы через интерфейс бота. Он будет искать ответы в проиндексированных веб-страницах.

## Зависимости

Основные библиотеки, используемые в проекте (подробнее см. в `requirements.txt`):
*   `langchain`: Фреймворк для создания приложений с LLM.
*   `faiss-cpu`: Библиотека для эффективного поиска по векторным эмбеддингам.
*   `openai` (или аналоги): Клиент для доступа к API языковой модели.
*   `beautifulsoup4` / `scrapy`: Вероятно, используются для парсинга веб-страниц (папка `scrap`).
*   `python-dotenv`: Для загрузки переменных окружения из файла `.env`.
